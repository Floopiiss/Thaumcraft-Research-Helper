# Thaumcraft Research Helper

## Usage

Pick the two aspects you want to connect with the **From** and **To**
dropdowns, then set **Min. Steps** to match the spacing you need in the
research note.

Click **Find Connection** to search for the shortest valid path that still
meets that minimum length.

If the result uses aspects you do not want, disable those aspects and try
again. The path may get longer, but the script will keep looking for a valid
route.
